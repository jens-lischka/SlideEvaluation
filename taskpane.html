How it works

Slide Evaluation extracts text and layout data from your slides and sends it to OpenAI's API for analysis. Your API key is stored locally in this browser only.

Criteria:
Layout — checks alignment, spacing, and visual hierarchy
Writing Style — checks grammar, tone, and consistency
Content Clarity — checks if the key message is clear
Accessibility — checks contrast, font sizes, and alt text
Focal Point — analyzes visual weight distribution
